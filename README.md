# Cyber Security Audit Script

## Description
A Bash script that collects basic Linux system information for security auditing.

## Features
- Current User
- Hostname
- Current Date
- Kernel Version
- Current Directory
- IP Address
- Disk Usage
- Memory Usage
- Logged In Users
- Top Running Processes

## Technologies Used
- Bash
- Kali Linux
- Git
- GitHub

## Run

chmod +x audit.sh
./audit.sh
